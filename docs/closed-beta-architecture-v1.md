# ForgePass LA Closed Beta Architecture v1

## Objective

Transform ForgePass LA from a concept platform into a real information platform using live data integrations.

---

# Core User Journey

User opens ForgePass LA

↓

Discover events

↓

Discover places

↓

Choose experience

↓

Open navigation

↓

Attend experience

---

# Closed Beta Scope

## Pillar 1 — Events

Capabilities

- Browse events
- Search events
- Filter events
- View event details
- Open ticket links

Candidate APIs

1. Ticketmaster Discovery API
2. Eventbrite API

Priority

Highest

---

## Pillar 2 — Discovery

Capabilities

- Attractions
- Restaurants
- Museums
- Nightlife
- Hidden gems

Candidate APIs

1. Google Places API
2. Yelp API

Priority

Highest

---

## Pillar 3 — Navigation

Capabilities

- Maps
- Directions
- Travel estimates
- Route launching

Candidate APIs

1. Google Maps
2. Mapbox

Priority

Highest
